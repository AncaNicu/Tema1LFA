##### Tipul aplicatiei

* Java Swing desktop application

##### 

##### Scopul aplicatiei

* verificarea respectarii unui pattern de catre un string, folosind expresii regulate



##### Instrumente folosite

* Java
* Eclipse IDE



##### Functionalitatile aplicatiei

* posibilitatea ca utilizatorul sa introduca string-ul si pattern-ul de la tastatura, intr-o fereastra cu campuri specifice celor doua date de intrare
* afisarea mesajului corespunzator verificarii, inclusiv tratarea erorilor
